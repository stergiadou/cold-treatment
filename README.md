# cold-treatment
Μέσα κρυοθεραπείας
Διαδραστικό βιβλίο, με περιγραφή των μέσων κρυοθεραπείας 
Η δημιουργία του βιβλίου έγινέ μέσω της πλατφόρμας bookcreator, ακολουθώντας τα βήματα δημιουργίας ηλεκτρονικού βιβλίου. 
Το βιβλίο συμπεριλαμβάνει μεθόδους και τεχνικές κρυοθεραπείας, τοπικά ή σε ολόκληρο το σώμα, για θεραπευτικούς σκοπούς. Η παρουσίαση γίνεται μέσω κειμένων, εικόνων και υπερσυνδέσμων. 
Στόχος του βιβλίου είναι να ενημερωθούν οι ενδιαφερόμενοι για την κρυοθεραπεία και να αποκτήσουν την ικανότητα επιλογής της κατάλληλης μεθόδου εφαρμογής κρυοθεραπείας.
